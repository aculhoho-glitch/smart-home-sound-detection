# Sound Event Detection with ExtraTrees

This project was developed for MLPC 2026 Task 5: The Challenge.

The goal is to detect domestic sound events in audio recordings and predict their temporal onsets and offsets.  
The final system uses an ExtraTrees classifier on pre-extracted acoustic features and converts segment-level predictions into event intervals.

## Task

The system predicts 15 domestic sound event classes, including:

- bell_ringing
- coffee_machine
- cutlery_dishes
- door_open_close
- footsteps
- keyboard_typing
- keychain
- light_switch
- microwave
- phone_ringing
- running_water
- toilet_flushing
- vacuum_cleaner
- wardrobe_drawer_open_close
- window_open_close

The prediction CSV format is:

```text
filename,annotation,onset,offset
```

## Final Results

| System | Non-hidden Test Macro F1 |
|---|---:|
| Decision Tree baseline | 0.3170 |
| Best ExtraTrees model | 0.4346 |

The best model was an ExtraTrees classifier with:

```text
n_estimators=100
max_depth=20
min_samples_leaf=1
max_features="sqrt"
class_weight="balanced"
```

Median filtering was tested as post-processing, but it did not improve performance.  
Therefore, the final submitted system uses the ExtraTrees model without median filtering.

## How to Run

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY_NAME.git
cd YOUR_REPOSITORY_NAME
```

Create an environment:

```bash
python -m venv .venv
```

Activate it on Windows:

```bash
.venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Start Jupyter:

```bash
jupyter notebook
```

Open:

```text
notebooks/challenge_solution.ipynb
```

Run the notebook to train/evaluate the model and generate:

```text
predictions_hidden_test_extratrees.csv
```
